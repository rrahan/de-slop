# de-slop

A Claude-Code skill that eliminates AI slop writing patterns from any text, making it sound more natural and human.

## Usage

Trigger this skill when writing, rewriting, editing, or reviewing content. It automatically detects and removes common AI-generated patterns like rhetorical questions, false ranges, and negative parallelism.


## Installation

Add the `SKILL.md` file to your Claude-Code skills directory.
